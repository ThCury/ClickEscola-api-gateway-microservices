O que é o Promtail? Agente coletor de logs da stack Grafana. Ele lê continuamente (faz o “tail”) a saída de cada container pelo socket do Docker, adiciona rótulos (container, status, rota) e envia as linhas para o Loki — é a ponte entre os logs crus e o Grafana.
📜
O Gateway (Nginx) e os serviços emitem os logs/traces; o Dozzle mostra os logs crus em tempo real.
Tudo orquestrado via Docker Compose. Só expõem portas ao host: Gateway :8080, Admin :8090, Grafana :3001 e Dozzle :9999. Os serviços de aplicação e o Cassandra ficam só na rede interna backend.
